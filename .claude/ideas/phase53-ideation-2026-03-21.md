# Phase 53 Ideation — Dashboard Polish, Nav Cleanup & Print Previews
**Date**: 2026-03-21

## Lighthouse Domain
Universal — applies to all verticals. Pre-pilot visual polish that doubles as landing page marketing material.

## Decision Rationale
Founder asked to "polish dashboards and work areas" with "razzmatazz" — make them beautiful, screenshot-ready for the landing page. Not a new domain phase, but a design cohesion phase. With 52 phases built and pilot imminent, this is the highest-leverage work: converting "feature-complete prototype" into "product someone loves looking at."

### Aesthetic Direction
- **Scoro-style** dashboard density and chart richness — data-dense, professional, rich visualizations
- **Less white space** — tighter composition than current layout
- Applied through existing **Signal Deck** dark slate palette (not Scoro's light theme)

### Sidebar Navigation Cleanup
Founder confirmed 4 changes:
1. Remove "Documents" from sidebar (org docs → Settings > Organization)
2. Rename "Delivery" → "Projects"
3. Move "Proposals" from Finance → Clients zone
4. Rename "Team & Resources" → "Team"

"Documents" sidebar item was a strawman concept — org-level compliance docs (BEE certs) that don't tie into any workflow. Founder agreed they belong in Settings.

### Document Preview Fix
Preview currently shows pages at half-size with normal fonts — not an accurate WYSIWYG. Fix: A4-proportioned container with CSS scale transform.

## Founder Preferences (Confirmed)
- Scoro layout/graph style but less white, applied to Signal Deck palette
- Screenshot-ready for landing page (16:9 desktop screenshots)
- All sidebar tweaks included
- Org docs moved to Settings > Organization
- Document preview must be print-accurate (A4 proportions)
- No new backend work — pure frontend
