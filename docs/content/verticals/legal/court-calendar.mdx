# Court Calendar

<Callout type="info">
  This feature is coming soon. The sections below describe planned capabilities.
</Callout>

## What's Planned

The court calendar will give law firms a dedicated view for tracking court dates, hearing schedules, motion deadlines, and filing due dates across all active matters. You will be able to see upcoming dates in a calendar or list view, set reminders for critical deadlines, and link each court date to the relevant matter and client. The calendar will support South African court jurisdictions, including the High Court, Magistrate's Court, and specialist tribunals.

Planned features include automatic deadline calculation based on court rules (for example, days to file a plea after service), conflict detection when multiple hearings overlap, and the ability to share calendar entries with team members assigned to the matter.

## What's Available Today

While the dedicated court calendar is in development, you can use the following features to track court-related dates:

- [Projects](/features/projects) -- create a project for each matter and use task due dates to track hearing and filing deadlines.
- [Customers](/features/customers) -- manage client records and link matters to the correct client.
- [Documents & Templates](/features/documents) -- store court documents and use templates for standard pleadings.
- [Notifications](/features/notifications) -- set up reminders for upcoming task due dates so nothing is missed.
