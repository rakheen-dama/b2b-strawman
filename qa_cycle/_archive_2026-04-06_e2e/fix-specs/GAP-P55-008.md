# Fix Spec: GAP-P55-008 — No Edit action for court dates

## Problem
Court dates in SCHEDULED or POSTPONED status have no "Edit" option in the Actions dropdown. Only Postpone/Cancel/Record Outcome are available. Users cannot modify details (date, time, court, judge, description) without changing status.

## Root Cause (confirmed)
- Frontend: `frontend/components/legal/court-date-list-view.tsx` line 50-63
  - `canTransition()` returns `canPostpone`, `canCancel`, `canOutcome` but has no `canEdit` flag
  - Lines 206-225: DropdownMenu only renders Postpone/Cancel/Record Outcome items — no Edit item
- Frontend: `frontend/app/(app)/org/[slug]/court-calendar/court-calendar-client.tsx` line 364-391
  - Detail panel action buttons also only show Postpone/Cancel/Record Outcome — no Edit button
- Backend: `CourtCalendarController.java` line 74-77
  - `PUT /api/court-dates/{id}` endpoint EXISTS and is functional
  - `CourtCalendarService.updateCourtDate()` validates mutable state (SCHEDULED or POSTPONED)
- No edit dialog component exists in the frontend.

The backend supports editing. The frontend just never built the Edit dialog or wired it into the Actions menu.

## Fix
1. Create `frontend/components/legal/edit-court-date-dialog.tsx`:
   - Similar to `CreateCourtDateDialog` but pre-populated with existing values
   - Calls `updateCourtDate(slug, id, data)` action on submit
   - Does NOT include `projectId` (cannot change the linked matter)
   - Fields: dateType, scheduledDate, scheduledTime, courtName, courtReference, judgeMagistrate, description, reminderDays

2. In `court-date-list-view.tsx`:
   - Add `canEdit` to `canTransition()` return: `true` for SCHEDULED and POSTPONED
   - Add `onEdit` prop to component interface
   - Add "Edit" menu item to dropdown (before Postpone)

3. In `court-calendar-client.tsx`:
   - Add `editTarget` state
   - Add "Edit" button to detail panel action buttons
   - Render `EditCourtDateDialog` with `editTarget`

4. The `updateCourtDate` action already exists in `court-calendar/actions.ts` (line 90-106).
