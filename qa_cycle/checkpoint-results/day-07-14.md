### Task creation

### UI interaction note
Radix UI dialog triggers (e.g., "Add Member", "Log Time" buttons) do not respond to Playwright's standard `.click()` method. The `dispatchEvent(new MouseEvent('click', ...))` workaround was required for all dialog triggers. Radix tab switching required `focus()` + `Enter` key. This is a Playwright/Radix compatibility issue, not a user-facing bug.
