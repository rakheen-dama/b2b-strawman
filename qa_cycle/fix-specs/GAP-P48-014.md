# GAP-P48-014: Resources page crashes with TypeError

**Severity**: major
**Status**: SPEC_READY
**Estimated effort**: 30 min

## Problem

The `/resources` page crashes with "Something went wrong" and a console TypeError: "Cannot read properties of null". The page is a Server Component that fetches two APIs:

1. `getTeamCapacityGrid(weekStart, weekEnd)` — returns `TeamCapacityGrid`
2. `api.get<Project[]>("/api/projects")` — returns project list

Both calls are wrapped in try-catch with fallback defaults, but the crash happens in child components when the API returns objects with null nested fields (e.g., `members[].weeks[].allocations` is null instead of `[]`).

## Root Cause Analysis

The `AllocationGrid` component (client component) accesses:
- `members[0]?.weeks.map(...)` — if `weeks` is null, `.map()` crashes
- `cell.allocations.length` in `CapacityCell` — if `allocations` is null, `.length` crashes
- `member.memberName.split(" ")` — if `memberName` is null, `.split()` crashes

The backend `CapacityGridService` may return null for these fields when no capacity records or allocations exist for a member.

Additionally, `api.get<Project[]>("/api/projects")` could return a paginated response `{ content: [...], page: {...} }` instead of a raw array, which would cause `.filter()` to fail.

## Fix

### Option A: Defensive null guards in child components (recommended)

**File: `frontend/components/capacity/allocation-grid.tsx`**
1. Guard `weekStarts` extraction: `const weekStarts = members[0]?.weeks?.map((w) => w.weekStart) ?? [];`
2. Guard `member.weeks` iteration: `{(member.weeks ?? []).map(...)}`

**File: `frontend/components/capacity/capacity-cell.tsx`**
1. Guard `cell.allocations`: `const allocs = cell.allocations ?? [];` then use `allocs` everywhere

**File: `frontend/app/(app)/org/[slug]/resources/page.tsx`**
1. Guard the grid response: after `grid = await getTeamCapacityGrid(...)`, add:
   ```ts
   grid = {
     members: (grid.members ?? []).map(m => ({
       ...m,
       weeks: (m.weeks ?? []).map(w => ({ ...w, allocations: w.allocations ?? [] })),
     })),
     weekSummaries: grid.weekSummaries ?? [],
   };
   ```
2. Guard the projects response: wrap `api.get` result with `Array.isArray()` check:
   ```ts
   const raw = await api.get<Project[] | { content: Project[] }>("/api/projects");
   const projects = Array.isArray(raw) ? raw : (raw as any).content ?? [];
   ```
